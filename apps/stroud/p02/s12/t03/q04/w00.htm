Symmetry about the axes
If only even powers of one variable exist the graph will be symmetric about the other variable axis.
The variable y only appears with an odd power, so there is no symmetry about the x-axis.
The variable x appears with both even and odd powers of x, so there is also no symmetry about the y-axis.
Crossing axes
When a curve crosses a variable axis the value of the other variable is 0.
The value x = 0 is not a permitted value of the independent variable because we cannot define division by zero.
Consequently, the curve does not cross the y-axis.
