Symmetry about the axes
If only even powers of a variable occur, the graph will be symmetric about the other variable axis.
Only even powers of y occur so the graph is symmetric about the x-axis.
The variable x only appears with an odd power so the graph is not symmetric about the y-axis
Crossing axes
When a curve crosses a variable axis the value of the other variable is 0.
When y = 0 then x = 0 so the curve crosses the x-axis at (0,0).
