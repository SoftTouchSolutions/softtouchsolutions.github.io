Test 7(c) Period, amplitude and phase difference
Period:
Since each trigonometric function repeats itself every 2π radians:
sec ( 3 θ  +  4 )
 =  sec ( 3 θ + 4 + 2 π )
 
 =  sec ( 3 θ + 2 π / 3 + 4 )
So the period is  2 π / 3.
Amplitude:
Since the amplitude of a periodic function is the difference between the maximum value and the average value of the output taken over a single cycle:
The maximum value of the output is ∞ so it is meaningless to talk of amplitude.
Phase difference:
The phase difference of a periodic function is the interval of the input by which the output leads or lags behind the reference function:
The phase difference relative to the reference function
g ( θ )  =  sec 3 θ  is a lead of 4 radians.
