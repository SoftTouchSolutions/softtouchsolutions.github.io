How to do this exercise
  • Exercises are divided into one or more steps, each of which contains one or more input fields.
  • Use the mouse or the keyboard to select an input field. You can select the fields in any order your like.
  • Enter the answer into the field using the keyboard or, where necessary, by 'picking up' a symbol from the special characters table.
  • When you have completed all the fields (or before, if you don't know some of the answers) click the 'Check your answers' button to evaluate your work.
  • If you have everything right you will be able to go on directly to the next step (or the next exercise, if you have completed all the steps).
  • If one or more input fields are incorrect you can make two attempts to get the correct answer(s). Click the 'Check your answers' button after each attempt.
  • If you have used up all your attempts and there are still incorrect fields the correct answers will be inserted into the fields.
Getting help
  • Click on the 'Show a hint' button to display a hint for the current step. Click the button again to hide the hint.
  • Click on the 'Show full working' button to display the full working for the entire exercise in a pop-up window.
  • Use the 'Tables of standard forms' if you need a reminder of the form of a particular expression.
  • Consult the relevant 'Frames' in the sixth edition of Engineering Mathematics. The frame references are given for each step of the calculation.
