The solutions to the quartic will give the x values of the stationary points.
The quartic can be written in nested form as
( ( ( x − 4 ) x + 2 ) x − 4 ) x + 5  =  0
This is satisfied if   x  =  1  so  x − 1  is a factor of the quartic.
