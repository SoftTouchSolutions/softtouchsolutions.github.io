Crossing axes
When the graph crosses a variable axis the value of the other variable is   0 .
When   x  =  0  the equation becomes   y − 2  =  0
so the graph crosses the  y -axis at  y  =  2
When   y  =  0  the equation becomes  x − 2  =  0
so the graph crosses the  x -axis at  x  =  2
