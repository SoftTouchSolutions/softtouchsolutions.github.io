Symmetry about the axes
If only  even  powers of  y  occur the graph will be symmetric about the  x -axis
y  only appears with an  odd  power so there is no symmetry about the   x -axis
If only  even  powers of  x  occur the graph will be symmetric about the  y -axis
x  only appears with both  odd  and even powers so there is no symmetry about the   y -axis
